## O tipo de dado que permite representar números de ponto flutuante é o "float".

Considerando: 

a = 5.0

b = 2.0

## Operadores Aritméticos
soma = a + b

subtracao = a - b

multiplicacao = a * b

divisao = a / b

## Operadores Aritméticos Compostos
a += 1.0  # Equivalente a a = a + 1.0

b *= 3.0  # Equivalente a b = b * 3.0
